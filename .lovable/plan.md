

# CORTYLIX TECHNOLOGIES — Full Website Build Plan

## Design System
- **Primary:** #10367D (deep blue)
- **Secondary:** #00B4D8 (cyan/neon blue)
- **Accent:** #051B3F (dark navy)
- **Background:** Dark theme with near-black (#0A0F1C) and dark gray tones
- **Text:** White/light gray
- **Font:** Clean sans-serif (Inter)
- Smooth animations, hover effects, futuristic feel

## Pages & Components

### Layout
- **Navbar** — Sticky, logo + nav links (Home, Services, Portfolio, Contact), "Request a Quote" CTA button, mobile hamburger menu
- **Footer** — Company name, quick links, social media icons, copyright © 2026

### Sections (Single-page scroll layout)

1. **Hero** — Bold headline, subtext about Tanzania, two CTA buttons, animated gradient/particle background
2. **About** — Company description, mission & vision with icons
3. **Services** — 4 cards (Web Dev, Software Dev, Graphic Design, IT Support) with icons, hover animations
4. **Why Choose Us** — 4 feature highlights (Fast delivery, Affordable, Local support, Reliable)
5. **Portfolio** — 6 project cards with placeholder images, titles, descriptions
6. **Testimonials** — 3 client reviews with names/roles
7. **Contact** — Form (name, email, message) with validation + contact info + WhatsApp button
8. **CTA Banner** — Persuasive text + "Request a Quote" button

### Extra Features
- **Floating WhatsApp button** — Fixed bottom-right
- **Request a Quote dialog** — Modal popup form
- **Language toggle** — English/Swahili switcher in navbar
- **Smooth scroll** navigation between sections
- **Mobile-first** responsive design throughout
